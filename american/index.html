John Dietz @ American Systems

PERFORMANCE SPECIALIST (Contractor) || American Systems, McLean VA

May 2011 – February 2013

John Dietz was brought to American Systems, a partner of the NG-CCAS program mentioned below, to single-handedly provide performance testing, analysis, and tuning for their Synchronized Predeployment Operational Tracker (SPOT) system. SPOT is web-based application that provides the capability to maintain accountability and report status for deployed personnel, and serves as the single source to track deployed contractor personnel supporting DoD military operations worldwide.

Mr. Dietz has helped produce a significantly improved online application to the end-user during his tenure. Since he joined the program in 2011, the SPOT application has accomplished a 65% reduction in average page time with increased system stability and a reduction in required hardware resources.

His responsibilities have included providing performance analysis and diagnosing performance bottlenecks, as well as providing architectural recommendations for the online Army system. He led the implementation of the dynaTrace analytics server to provide retroactive visibility to CLR content across 14 lab environments. He has administered both the analytics server and the Load Test Rig. He has led the effort to transition American System’s load testing technology from Load Runner to Visual Studio and significantly reduced project licensing costs. Technologies have included .NET 3.5, SQL Server 2008 R2, SSRS, IIS, WCF Web Services, WSE 2.0, TFS, dynaTrace Server, dynaTrace AJAX edition, and Visual Studio 2010 Web and Performance Testing.
